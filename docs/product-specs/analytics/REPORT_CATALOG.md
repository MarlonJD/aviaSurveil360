# Report Catalog

## MVP reports

### Audit Report

Audience: Inspector, manager, auditee.

Contains: scope, organization, inspectors, checklist summary, findings, observations, due dates, CAP requirements, evidence status, issue date.

### Open Findings Report

Audience: Inspector, manager.

Contains: finding number, organization, severity, due date, current owner, next action, status.

### Overdue Findings Report

Audience: Manager.

Contains: overdue findings, days overdue, severity, organization, inspector, auditee contact, escalation status.

### Organization Findings Report

Audience: CAA and auditee, filtered by visibility.

Contains: organization findings, open/closed status, CAP status, evidence status, closure history.

## Phase 2 reports

## Visibility rule

Auditee report versions must exclude internal notes, inspector performance, other organizations, internal risk calculation and draft enforcement notes.
