# Audit Checklist Workflow — Türkçe

## Amaç

Help inspectors run checklists and create findings directly from non-compliant answers.

## Adımlar

1. Open audit
2. Start checklist
3. Answer question
4. Add comment/evidence
5. Create finding if non-compliant
6. Complete section
7. Complete checklist
8. Generate draft report

## Kurallar

- Answers: Compliant, Non-Compliant, Observation, Not Applicable, Not Checked
- Non-Compliant offers Create Finding
- Completed checklist locked unless reopened with permission

## UX notları

- Current owner, due date ve next action ekranın üstünde gösterilmeli.
- History timeline/tab içinde kalmalı, primary content olmamalı.
- Primary button next action ile aynı dili kullanmalı.
