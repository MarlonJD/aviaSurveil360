# Navigation ve Information Architecture

## Inspector IA

Home genel dashboard değil, workbench olmalı.

Top queues:

- Today's Inspections
- Checklists In Progress
- CAPs Waiting Review
- Evidence Waiting Review
- Due Soon
- Overdue

## Auditee IA

Auditee sadece kendi organization'ını ve kendisinden beklenen aksiyonları görmeli.

Top queues:

- CAP Required
- Evidence Required
- More Information Requested
- Due Soon
- Overdue
- Closed Findings

## Manager IA

Manager dashboard exposure ve delay göstermeli.

Top cards:

- Oversight Health Index
- Open Findings
- Overdue Findings
- Critical Findings
- Plan Completion
- Average Closure Time

Needs Attention list:

- Level 1 findings open
- Overdue Level 2 findings
- Repeated findings by organization
- CAPs rejected multiple times
- Audits not started as planned

## Admin IA

Admin configuration yönetmeli, daily work değil.

Top areas:

- Checklist Templates
- Audit Types
- Organization Types
- Severity Rules
- Due-Date Rules
- Notification Templates
- Roles & Permissions
- Audit Log
