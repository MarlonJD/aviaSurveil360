## Demo Boundary

AviaSurveil360 remains a planning pack plus a frontend-only static clickable
demo. The docs do not claim backend, database, API, real authentication, real
authorization enforcement, real upload/storage, real AI service, real
regulatory ingestion, real notification delivery, production audit-log
behavior, remote CI, or production readiness.
